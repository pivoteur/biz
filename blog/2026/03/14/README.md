# Automation

## `gecko`

G'day, pivoteurs!

![`gecko` automation](imgs/01-gecko.png)

Our dev, @ParisBrand32180, has automated `gecko`, the dapp that fetches the 
token-prices and populates those data into the Pivot Protocol.

This automation saves me 1 hour/day.

AND it also creates a pathway for us to automate more of the Pivot Protocol.

## `virtsz`

What's next for us for automation?

Managing virtual-pivots manually is onerous: I've done 3 virtual updates in 
the last 3 years! No joke.

With `virtsz` – the virtualization dapp – the Pivot Protocol could manage 13 
virtual pivots in seconds!

`virtsz` is currently in β-test.
